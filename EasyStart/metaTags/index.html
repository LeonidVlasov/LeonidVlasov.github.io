Метатеги - это теги, содержащие в себе информацию, которая предназначена для браузеров, поисковых систем, программного обеспечения и сервисов, которые так или иначе могут взаимодействовать с веб-приложением (сайтом).

Такие теги могут нести в себе описание страницы (например, ключевые слова и само описание), информацию об авторских правах или самом авторе, указания браузеру на какие-то внешние или внутренние механизмы обработки страницы и даже команды браузеру, а также правила «поведения» для поисковых систем.

Мета теги должены находиться внутри head и быть в пределах 1024 первых байтов страницы, так как некоторые браузеры смотрят только на эти байты, перед тем как определить набор символов страницы.

Атрибуты мета тегов

charset Стал поддерживаться с HTML 5. Этот атрибут задает кодировку символов, используемую на странице. Может быть локально переопределен. Рекомендуется использовать кодировку UTF-8.
content Этот атрибут даёт значение, связанное с http-equiv или name атрибутами, зависящее от контекста.
http-equiv Этот атрибут определяет поведение серверов и пользователей. Значение задается с помощью content и может быть следующим:
Content-Security-Policy Это значение позволит администратору веб-сайта определить политику содержания для обслуживаемых ресурсов. За некоторыми исключениями, политика в основном включают в себя указание происхождения сервера и конечные точки сценария. Это помогает предотвратить атаки межсайтового скриптинга.
Default-style Специализация этой прагмы - предпочтительный стиль таблиц, используемый на странице. content атрибут должен содержать заголовок link элемента который href связывает атрибут с CSS таблцей стилей, или заголовок style элемента, который cодержит CSS таблицу стилей.
Refresh Эта прагма определяет: Количество секунд перезагрузки таблицы, если content атрибут содержит только положительный целочисленный номер; Время, в количестве секунд, за которое страница должна быть перенаправлена ​​на другую, если content атрибут содержит положительный целочисленный номер, заканчивающийся строкой 'url=' и корректный URL.
name Определяет метаданные, значения для которых задаются атрибутом content. Существуют следующие распространненые значения:
application - name определяет имя веб-приложения, запущенного на веб-странице
author/creator - определяет в свободном формате имя автора документа.
description - краткое и точное резюме содержания страницы. В некоторых браузерах, среди которых Firefox и Opera, этот мета используется как описание страницы по умолчанию в закладке/span>
generator - указывает имя программного обеспечения, которое использовалось при генерации данной страницы.
keywords - указывает ключевые слова страницы (используется поисковыми системами при анализе страницы)
robots - определяет поведение, поисковых роботов нас транице. Значения для содержимого meta name="robots"
index Позволяет роботу индексировать страницу
noindex Освобождает робота от идексирования страниц
follow Позволяет роботу переходить по ссылкам со страницы
nofollow Запрещает роботу переходить по ссылкам со страницы
none Эквивалетно noindex, nofollow
googlebot - аналог robots, но только следует за Googlebot, сканирует индексы для Google;
slurp - аналог robots, но следует только за Slurp, индексирующим роботом от Yahoo Search;
publisher - определяет в свободном формате имя того, кто опубликовал документ
viewport - дает подсказки о размере изначального размера viewport. Эта прагма используется только на некоторых мобильных устройствах. Значения для содержимого meta name="viewport"
width - Определяет ширину области просмотра в пикселях
height - Определяет высоту области просмотра в пикселях
initial-scale - Определяет соотношение между шириной устройства и размером области просмотра
maximum-scale - Определяет максимальное значение зума
minimum-scale - Определяет минимальное значение зума
user-scalable - (доступны значения - да или нет) если нет, то пользователю не доступен зум на веб-странице. По умолчанию задано значение да. Cпециальные значение от соц сетй - для более корректной работы с роботами соц сетей
Cпециальные значение для соц сетей - для более корректной работы с роботами соц сетей
scheme НЕ ПОДДЕРЖИВАЕТСЯ В HTML 5. Указывает полезную информацию о схеме или название самой схемы, которая должна быть использована для интерпретации значения свойства
